### 目录结构

```
├── content/post/       ← 所有文章都扔这里，按年份月份乱糟糟放着
├── static/img/         ← 文章封面图、头像、表情包请放这儿
├── config/_default/    ← 改网站名字、换头像、挂友情链接的地方
└── themes/stack/       ← 主题，没事别乱动（血的教训）
```
